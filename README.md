# 🔒 Güvenli Dosya Şifreleme Uygulaması

Bu proje, Veri Güvenliği dersi kapsamında kriptografik algoritmalar kullanılarak, hassas verilerin güvenli bir şekilde saklanması ve iletilmesini sağlamak amacıyla geliştirilmiştir.

## 🚀 Özellikler
* **Gelişmiş Şifreleme:** Dosyaları güvenli şifreleme algoritmaları ile şifreler ve geri çözer.
* **Anahtar Yönetimi:** Güvenli anahtar (key) üretimi ve doğrulaması sağlar.
* **Veri Bütünlüğü:** Şifreleme esnasında dosya yapısının bozulmasını önler.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Python
* **Kütüphaneler:** Cryptography, OS, PyCrypto 

## 💻 Kurulum ve Çalıştırma
1. Projeyi bilgisayarınıza indirin:
   ```bash
   git clone github.com
   ```
2. Uygulamayı çalıştırın:
   ```bash
   python main.py
   ```
